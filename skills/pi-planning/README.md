# PI Planning

- PI Objectives
- Capacity Plans
- Feature Breakdown
- Risks & Dependencies

## Inputs

- Strategic Objectives
- Team Capacity
- Feature Backlog
- Cross-Team Dependencies

## Outputs

- PI Objectives
- Sprint Allocation
- Risks
- Dependencies
- Executive Summary
